  • 下载后解压，在解压目录中 右键 → 以管理员身份运行 Clash for Windows.exe。

软件使用教程 ​

步骤 1：一键导入订阅链接 ​

  1. 进入 官网仪表盘，点击 “一键订阅”。
  2. 点击 Windows，再点击 导入到Clash for Windows。
  3. 点击 打开Clash for Windows 按钮等待导入完成

步骤 2：选择节点 ​

  1. 点击 Clash 面板左侧的 [代理]。
  2. 选择上方的 [规则] 选项。
  3. 在 [节点选择] 选项卡下，选择你要使用的线路。

重要提示

❌ 切勿选择 [全局] 或 [直连]，否则可能导致无法访问外网。

步骤 3：连接代理 ​

  1. 点击 侧边栏 [常规]。
  2. 勾选 "系统代理" 即可启用。
  3. 如果无法使用，请尝试 更换节点（详见下方换节点教程）。

检查是否成功连接 ​

浏览器输入 youtube.com 检查是否正常使用

进阶使用方法 ​

开启服务模式 ​

  1. 点击[服务模式] 后面的 管理 按钮。
  2. 点击 安装，等待安装完成。
  3. 安装成功后，左侧的小地球图标会变成 绿色 ✅，表示已启用。

开启 TUN 模式（加速游戏） ​

打开 [TUN模式] 开关，即可对游戏进行加速。

重要提示

⚠ 注意：TUN 模式会代理电脑所有流量！

  • 适用于 游戏加速，但可能影响本地网络。
  • 建议：用完后立即关闭，避免不必要的全局代理流量消耗。

常见问题（FAQ） ​

Clash for Windows 无法一键导入，如何手动导入配置？

步骤 1：获取订阅链接 ​

  1. 进入 官网仪表盘，点击 “一键订阅”。
  2. 点击 “复制订阅链接”。

步骤 2：导入订阅链接 ​

  1. 打开客户端，进入 [配置] 页面。
  2. 粘贴订阅链接 到上方 URL 输入框。
  3. 点击 [下载]，完成导入。

订阅失败排查

  • 检查网络是否正常
  • 不要开启系统代理以及Tun模式
  • 确认订阅地址是否完整
  • 尝试手动输入 URL 而非粘贴
启用 [系统代理] 后，UWP 应用无法联网？
  1. 选择侧边栏 [常规]。
  2. 点击 [UWP应用网络回环]。
  3. 在打开的应用列表中，勾选对应的 UWP 应用。
  4. 点击列表上方的 Save 保存设置。
Clash for Windows 如何切换节点？
  1. 选择侧边栏 [代理]。
  2. 点击 展开 Proxy 分组。
  3. 在 节点选择 分组下选择 非超时的节点，即可完成切换。
Clash for Windows 如何判断节点是否超时？
  1. 选择侧边栏 [代理]。
  2. 点击 Proxy 分组旁边的 📶 (WiFi) 按钮 进行延迟测试，如果是红色字体则为超时。
Clash for Windows 如何更新节点？
  1. 选择侧边栏 [配置]。
  2. 在 Bluetile.biz 右侧点击 🔄 (刷新) 按钮，即可更新节点。

重要提示

部分节点显示 超时 仍然可用，建议多次测试后选择 较稳定的节点。
